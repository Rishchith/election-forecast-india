# Election Forecasting Model (India)

## Objective
Predict constituency-level winners using data instead of surveys.

## Method
- Historical vote share
- Swing factor
- Alliance strength
- Demographics
- Turnout behavior

## Model
Feature-weighted scoring system:
Score = weighted sum of features → classify → pick highest

## States Covered
- Kerala
- Tamil Nadu
- Assam
- West Bengal

## Key Insight
Seat conversion matters more than vote share in FPTP systems.

## Limitations
- No booth-level data
- No real-time survey data
- Assumes semi-uniform swing
